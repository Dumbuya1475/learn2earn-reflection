# Learn2Earn - Tell Your Story Submission

## 👋 Overview
This repository contains my submission for the Learn2Earn bounty program. It represents my learning journey, challenges faced, and progress made while participating in the program.

## 🎯 Focus / Niche
Frontend Web Development using React and modern web technologies, with a focus on building real-world, freelance-ready projects.

## 📚 What I Learned
- How to complete structured tasks under deadlines
- How to work like a freelancer (build, deliver, submit)
- Importance of consistency and finishing tasks instead of delaying work
- Basic project structuring and deployment workflow

## ⚠️ Biggest Challenge
My biggest challenge was consistency and managing deadlines. I often delayed tasks until close to submission time, which affected my workflow and increased pressure.

## 🛠️ How I Improved
I started breaking tasks into smaller steps, working earlier instead of waiting for deadlines, and treating each task like a real client project. This helped me become more consistent and improved my delivery quality.

## 🚀 Next 3 Months Plan
- Build 2–3 real-world projects for my portfolio
- Improve my React and frontend development skills
- Practice freelancing-style project delivery
- Focus on consistency and shipping work regularly
